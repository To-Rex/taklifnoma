# TaklifNoma - Wedding Invitation Platform

A comprehensive wedding invitation platform built with React, TypeScript, Supabase, and Tailwind CSS. Create beautiful, customizable wedding invitations with real-time RSVP tracking.

## 🌟 Features

### ✅ **Complete Authentication System**
- Google OAuth Sign-in
- Email/Password authentication
- Protected routes and pages
- User profile management
- Session persistence

### 🗄️ **Real Database Integration**
- PostgreSQL database with Supabase
- Real-time data synchronization
- Secure Row Level Security (RLS)
- Backup localStorage fallback

### 🎨 **Advanced Template System**
- 15+ professional wedding templates
- Custom template builder
- Real-time design preview
- Color, font, and layout customization
- Public/private template sharing

### 📱 **Modern User Experience**
- Fully responsive design
- Beautiful animations throughout
- Real-time form validation
- Loading states and error handling
- Success feedback

### 💌 **Invitation Management**
- Create and customize invitations
- Real-time RSVP tracking
- Guest list management
- Multiple sharing options (WhatsApp, Telegram, etc.)
- PDF export functionality

### ⚙️ **Complete Settings System**
- Comprehensive user profiles
- Privacy and security settings
- Notification preferences
- Account management
- Data export functionality

## 🛠️ **Technical Stack**

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Custom animations
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **State Management**: React Context
- **Routing**: React Router 6
- **UI Components**: Custom component library
- **Icons**: Lucide React

## 📊 **Database Schema**

### Tables Created:
- `profiles` - User profile information
- `invitations` - Wedding invitation data
- `guests` - Guest list management
- `rsvps` - RSVP responses
- `custom_templates` - User-created templates

### Security:
- Row Level Security (RLS) enabled
- User-specific data isolation
- Secure authentication policies
